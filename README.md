# Simple Shell

## Description

Simple Shell is a UNIX command-line interpreter written in C. It is a simplified version of the standard UNIX
shell (/bin/sh) developed as part of the ALX Software Engineering Program.

The shell reads commands from standard input, creates child processes to execute them, and waits for the processes to
finish before displaying the prompt again. It supports both interactive and non-interactive modes.

This project demonstrates fundamental concepts of operating systems and systems programming, including process creation,
process control, command execution, environment handling, and inter-process communication.
